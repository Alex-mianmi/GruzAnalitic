# Marketing Intelligence Presentation

Интерактивная сайт-презентация маркетинговых аудитов:

- `GruzTime · gruztime.com`
- `ТРАНССКАЙ · dostavka177.com`
- `5TONES · 5tones.ru` — экран ожидания до добавления аудита

## Запуск

Откройте `index.html` напрямую или запустите локальный сервер:

```bash
python3 -m http.server 8000
```

Затем откройте `http://localhost:8000`.

## Публикация

При каждом push в ветку `main` сайт автоматически публикуется через GitHub Pages.

## Навигация

- Верхние вкладки переключают сайты.
- Боковое меню переключает главы.
- Стрелки клавиатуры, `PageUp` и `PageDown` листают презентацию.
- Кнопка «Презентация» включает полноэкранный режим.

Все данные и содержание слайдов находятся в `app.js`.
